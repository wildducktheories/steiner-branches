# Steiner Branches Lattice Visualizer

An interactive web application for visualizing the Steiner branch formulas A(n,x), B(n,x), and C(n,x) for the Collatz map.

## Overview

This SPA (Single Page Application) displays an (n,x) lattice where each lattice point represents values generated by the branch formulas:

- **A(n,x)**: First branch formula (marked with `-`)
- **B(n,x)**: Second branch formula (marked with `+`)
- **C(n,x)**: Common endpoint formula

The visualization shows how these formulas classify odd integers according to their Steiner circuit structure in the Collatz map.

## Features

- **Interactive Lattice**: Click on any point to see its Collatz trajectory
- **Axis Configuration**: n on x-axis, x on y-axis (since x can be large)
- **Logarithmic Scale**: Optional log₁₀ scale for x-axis to handle large x values
- **Branch Visualization**: Points colored by branch (A = blue, B = red)
- **Connection Mapping**: Shows Collatz map connections between odd terms and endpoints
- **Tooltips**: Hover over points to see detailed information
- **Formula Reference**: Built-in reference for A(n,x), B(n,x), C(n,x) formulas

## Branch Formulas

### A(n,x)
```
A(n,x) = {
  3·2^(n-1) + 2^(n+1)·x - 1,  if n odd
  2^(n-1) + 2^(n+1)·x - 1,    if n even
}
```

### B(n,x)
```
B(n,x) = {
  3·2^n + 2^(n+2)·x - 1,  if n odd
  2^n + 2^(n+2)·x - 1,    if n even
}
```

### C(n,x) - Endpoint
```
C(n,x) = 2·3^n·x + 4·Σ(i=0 to ⌊(n-1)/2⌋) 9^i
```

## Usage

1. Open `index.html` in a modern web browser
2. Set lattice parameters:
   - **Max n**: Maximum n value for the lattice (1-15)
   - **Max x**: Maximum x value for the lattice (0-100)
   - **Start value**: (Optional) Starting value to trace Collatz trajectory
   - **Log scale**: Enable logarithmic scale for x-axis (useful for large x values)
3. Click "Generate" to render the lattice
4. Click on any lattice point to see its Collatz trajectory
5. Hover over points to see detailed information

**Note**: The lattice is displayed with n on the x-axis and x on the y-axis, since x can grow large while n typically stays small.

## Files

## Technical Details

- Uses BigInt for accurate large integer calculations
- Canvas-based rendering for performance
- No external dependencies - pure JavaScript/HTML/CSS

## Related Work

This visualizer is based on the mathematical framework described in:

**"Branch Formulas for the Collatz Map: A First-Principles Derivation via Steiner Circuits"**
by Jon Seymour

The paper presents a complete derivation of the branch formulas from Steiner circuits without relying on global convergence claims.
